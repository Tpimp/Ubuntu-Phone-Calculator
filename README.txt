Ubuntu-Phone-Calculator
=======================

Touch driven calculator with advanced functions. Built using Qt5 and the Bignum C++ library (ttmath.org)

This project was originally derived from an effort to design and realize a full Ubuntu Phone Calculator application.
(https://wiki.ubuntu.com/Touch/CoreApps?action=show&redirect=UbuntuPhone%2FCoreApps)

All designs, development(code), and art is created by Christopher Dean.
